## Proceso
En este proyecto se han incluido los siguientes elementos y tecnologías:
- Eventos `onclick` en botones para interactividad.
- Evento `onload` para inicializar ciertas funciones al cargar la página.
- Bucles (`forEach`) para recorrer arrays y mostrar datos.
- Condicionales (`if`) para la lógica de las funciones.
- Uso del objeto `Date` para la lógica basada en el tiempo.
- Almacenamiento local (`localStorage`) para guardar y cargar comentarios.
- Input y button para la interacción del usuario.

## Reflexiones
Esta actividad fue muy divertida y desafiante. Al no utilizar frameworks ni librerías externas, me permitió reforzar conocimientos en HTML, CSS y JavaScript puro, lo cual me pareció muy útil. A continuación, se presentan algunas reflexiones:
- Con HTML, CSS y JavaScript se pueden crear sitios web rápidos y eficientes.
- La página completa pesa menos de 1MB, lo cual es ideal para una carga rápida en cualquier dispositivo.
